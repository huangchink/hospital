### Create conda environment

conda create --name envname python=3.12
conda activate envname

### Installing package
```bash
pip install screeninfo
pip install pandas
pip install MNN==3.1.3
pip install opencv-python
pip install pygame
pip install mediapipe
```

### run setup.py 
```
python setup.py install
```
